# ECE362 Final Project Smart Mirror

## Description
The Smart Mirror is an embedded system designed to provide an interactive display integrated into a mirror surface. The system presents useful daily information such as the current time and an alarm while maintaining the appearance of a standard mirror when not in active use. The device will utilize multiple microcontroller peripherals to manage user input, display output, and sensor data.

The goal of this project is to demonstrate the integration of several embedded peripherals and communication protocols to build a responsive and modular user interface.

The 4 peripherals we will be using include HSTX for display, DAC for audio, PWM for volume control, and ADC for LED brightness. 

## 362 Project Objectives
- An ability to use HSTX to output video to an HDMI monitor.
- An ability to use PIO to control WS2812 LED strips.
- An ability to use ADC to read a potentiometer to control the volume of the speaker
- An ability to use PWM to generate an alarm sound
